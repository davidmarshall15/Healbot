Reset HealBot


To set absolute defaults for Healbot on all Accounts and Realms, do the following in order:

1: Completely Logout of WoW
2: Download Reset_HealBot.zip

On Windows
3: Unzip Reset_HealBot.zip to a temporary folder
4: Copy Reset_HealBot.bat into the World of Warcraft folder (ie. C:\Program Files\World of Warcraft)
5: Run Reset_HealBot.bat
6: Type “D” to delete Healbot’s WTF files
7: That’s it, logon to WoW and set Healbot Spells, Buffs, Cures and Skins


On Mac
3: Unzip Reset_HealBot.zip to a temporary folder
4: Copy Reset_HealBot.pl into the World of Warcraft folder (ie. /Applications/World of Warcraft)
5: Run Reset_HealBot.pl using the Terminal application located in the Applications/Utilities Folder, and type: perl /Applications/'World of Warcraft'/Reset_HealBot.pl
6: Type “Y” to delete Healbot’s WTF files
7: That’s it, logon to WoW and set Healbot Spells, Buffs, Cures and Skins
